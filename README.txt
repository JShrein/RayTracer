Instructions:
In any platform, you should check in Tracer_2/World.cpp at lines 56 and 57.  If you are using Windows, set USEWIN to 1.  If you are using MPI, set USEMPI to 1

On Linux, compile using:
make -f Makefile_Linux all

On Linux using MPI, compile using:
make -f Makefile_Linux mpi-all

On Mac, compile using:
make -f Makefile_mac all

On Windows, project was built using Visual Studio 2013 Express
Open Tracer_2.sln to load project into Visual Studio 2013 or greater
